Forms without the paperwork

You don't have to add users by filling in form data by hand. You can bind data to the form programmatically. See views.automatic for the details.

To insert members from a list of seven usernames, just click the button below. Note that both the automatic and manual methods use utils.add_user to add the form data to the database. The only difference is how the form is bound. Note I purposely have left it so you can enter the same user multiple times: this is not how you would want things IRL.

Obviously, this this programmatic data entry is the kind of thing you would want to do in the background using cron or celery. This project is only meant to show the types of tasks you would automate, but the automation itself (for instance updating a clan list) is left out for now because I don't know how to do it yet.
